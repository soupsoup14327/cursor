# Anime Arena

Мини-игра в аниме-стиле на чистом HTML/CSS/JS.

## Что есть

- выбор персонажа (3 героя)
- пошаговый бой против босса
- действия: обычная атака, спецприем, блок
- лог боя, раунды, комбо и счетчик побед
- стилизованный интерфейс и анимации попаданий

## Как запустить

1. Открой `index.html` в браузере.
2. Выбери героя.
3. Сражайся с боссом и перезапускай бой кнопкой "Новый бой".

## Ассеты

В текущей версии используются встроенные SVG-аватары (генерируются в `script.js`), поэтому проект запускается без загрузки сторонних файлов.

Если хочешь, можно заменить их на внешние ассеты в свободной лицензии (CC0 / Public Domain / CC BY) и подставить ссылки или локальные файлы в поле `avatar` у героев.
