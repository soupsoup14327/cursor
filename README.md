# Module battle

Пошаговая сцена в духе настольного D&D: выбери героя, сразись со случайным врагом из «модуля», веди журнал столкновения. Чистый HTML, CSS и JS — без картинок-аватаров (сигилы с буквами).

## Запуск

1. Открой `index.html` в браузере **или** в корне проекта: `npm start` (Electron).
2. Сборка Windows `.exe` (portable): `npm install` затем `npm run build` — результат в папке `dist/`.

## Механики

- три героя с разными наборами из пяти приёмов;
- **мана**, перезарядки, случайный враг из пула;
- перед вашим ходом и ходом врага бросаются **d20 и d8** — бонус к урону считается из суммы кубиков;
- **статусы**: огонь, яд, кровотечение (урон в конце раунда), мороз (ослабляет следующий по вам удар);
- случайные **всплывающие сценарии** (скользнули, прилив сил и т.п.);
- на мобильных карточки «герой — VS — враг» в одну линию;
- атмосферная строка в начале боя и подсказка мастеру после каждой третьей победы подряд.
